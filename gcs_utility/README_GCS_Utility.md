# GCS Utility

### Pre-requisite:
1. Install VENV and Activate it
    ```shell
    python -m venv venv
    source venv/bin/activate
    ```
2. Install Required Packages:
    ```shell
    pip install google-cloud-storage
    ```

### Sample Code:
1. Print Bucket Lists and Blob lists
    ```shell
    python gcs_app.py
    ```
